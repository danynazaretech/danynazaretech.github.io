Acessibilidade para todos

O AssistiView SDK permite que desenvolvedores de Kotlin integrem suporte compatível TalkBack e às APIs de acessibilidade do Android em poucas linhas de código — tornando seus apps acessíveis a pessoas com deficiencia visual e com baixa visão de forma dinâmica e eficiente.

Open Source
WCAG 2.1 AA
Android API 21+
Kotlin First
Instalação

Pronto para usar em minutos

Disponível via Maven Central. Compatível com Android API 21+ e Kotlin 1.8+.

build.gradle.kts 
implementation("io.assistiview:sdk:1.0.0")

// Opcional: extensões para Jetpack Compose
implementation("io.assistiview:sdk-compose:1.0.0")
pom.xml 
<dependency>
    <groupId>io.assistiview</groupId>
    <artifactId>sdk</artifactId>
    <version>1.0.0</version>
</dependency>
bash 
git clone https://github.com/assistiview/sdk-android.git

cd sdk-android

./gradlew publishToMavenLocal

Como funciona

Integre em 4 passos simples

Do zero à acessibilidade completa em menos de 5 minutos.

01

Adicione a dependência

Inclua o AssistiView SDK no seu arquivo build.gradle.kts com uma única linha.

implementation("io.assistiview:sdk:1.0.0")
02

Inicialize no Application

Configure o SDK na sua classe Application com as opções desejadas.

AssistiView.init(this) { debugMode = true }
03

Anote suas Views

Use as extensões Kotlin para adicionar descrições e comportamentos.

myButton.assistiView { contentDescription("Enviar") }
04

Teste com TalkBack

Ative o TalkBack e navegue pela sua interface.

AssistiView.audit(rootView) { issues -> ... }
Recursos

Tudo que você precisa para construir com inclusão

O AssistiView SDK oferece uma camada de abstração sobre as APIs nativas do Android, tornando a acessivel navegar em telas sensíveis ao toque para pessoas com deficiência visual ou com baixa visão. Digitar em telas sensíveis ao toque é difícil para pessoas com necessidade visual. A bilioteca AssistiView introduz uma nova abordagem baseada em gestos direcionais, permitindo que usuários utilizem movimentos simples direcionais para explorar a tela. Cada interação fornece feedback tátil (vibração) e auditivo, garantindo uma experiência de exploração dos elementos da mais acessível, eficiente e intuitiva.

Acessibilidade

TalkBack Nativo

Integração direta e compatibilidade com o serviço de acessibilidade TalkBack do dispositivos móveis Android. Ações customizadas através de uma lógica de navegação dinâmico por gestos .

Developer Experience

API Kotlin-First

Construído especificamente para Kotlin, com suporte para coroutines, funções de extensão e um DSL fluente. Integre-o perfeitamente à sua interface de usuário existente baseada em visualização com apenas algumas linhas de código..

UX Acessível

Navegação Dinâmica

Defina a ordem de foco, grupos lógicos e hierarquias de navegação de forma programática. Usuários cegos navegam sua UI de forma fluida e previsível.

Conformidade

WCAG 2.1 AA

Componentes e helpers que garantem conformidade com as diretrizes internacionais de acessibilidade web e mobile. Auditoria integrada em modo debug.

Compatibilidade

Suporte a Views e Compose

Funciona tanto com o sistema de Views tradicional quanto com Jetpack Compose. Migre gradualmente a árvore de elementos da interface sem reescrever todos os seus.

Performance

Leve e Sem Dependências

O SDK adiciona menos memória ao seu APK e não interfere com outras bibliotecas de UI ou frameworks de acessibilidade.

Open Source

Construído pela comunidade, para a comunidade

O AssistiView SDK é open source e aceita contribuições. Reporte bugs, sugira melhorias ou envie pull requests no GitHub.
